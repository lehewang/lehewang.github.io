Markdown基本语法

发表于 2020-01-14 | 分类于 工具手册 | 阅读次数: | 本文总阅读量 次

1. 功能快捷键

撤销：Ctrl/Command + Z
重做：Ctrl/Command + Y
加粗：Ctrl/Command + B
斜体：Ctrl/Command + I
标题：Ctrl/Command + Shift + H
无序列表：Ctrl/Command + Shift + U
有序列表：Ctrl/Command + Shift + O
检查列表：Ctrl/Command + Shift + C
插入代码：Ctrl/Command + Shift + K
插入链接：Ctrl/Command + Shift + L
插入图片：Ctrl/Command + Shift + G

2. 标题创建

写文章嘛，总少不了标题。Markdown支持6级标题，创建的方法也简单：

一级标题：输入一个#,然后输入一个空格，然后输入标题名称

二级标题：输入两个#,然后输入一个空格，然后输入标题名称

以此类推，1~6级标题只不过是#的个数不同。

4.1 插入链接

语法如下：

5. 代码

5.1 插入代码块

语法：在代码的起始和末尾添加```或者~~~，可以配置编程语言。

示例：

```python
print(‘This is python code’)
```

5.2 在文本中插入短代码

语法：在代码起始和末尾添加`。

示例：

​ 本章主要介绍`print()`函数。

效果如下：

​ 本章主要介绍print()函数。

6. 数学表达式

6.1 数学表达式块

语法：输入两个$，然后回车，即可输入数学表达式块。

示例：

$$

y = x + 1

$$

效果如下：
$$
y = x + 1
$$

6.2 嵌入数学表达式

语法：在公式前后各加一个$.

示例：

我要在这一行添加一个公式$y=x+1$。

效果如下：

我要在这一行添加一个公式$y=x+1$。

8. 表格

示例：
| 姓名 | 性别 | 毕业学校 | 工资 |
| —- | —- | ——– | —- |
| 杨洋 | 男 | 重庆大学 | 3200 |

9. 列表

9.1 有序列表

语法：在每个列表项的前面添加-和空格(空格不可以省略）

示例：

- 无序列表项1

- 无序列表项2

效果如下：

  • 无序列表项1
  • 无序列表项2

9.2 无序列表

语法：在列表项的前面添加数字.和空格。

示例：

1. 有序列表项1
2. 有序列表项2
3. 有序列表项3

效果如下：

  1. 有序列表项1
  2. 有序列表项2
  3. 有序列表项3

9.3 任务列表（复选框）

语法：

-空格[空格]空格文字：表示没有选中的复选框

-空格[x]空格文字：表示选中的复选框

示例：

- [ ] 计划任务
- [x] 完成任务

效果如下：

  • 计划任务
  • 完成任务
